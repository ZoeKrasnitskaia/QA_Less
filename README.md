# Автотесты для проекта

Репозиторий содержит набор автоматических тестов на базе фреймворка `pytest`.

## Быстрый старт (Инструкция по запуску)

Чтобы развернуть проект локально и запустить тесты, выполните следующие шаги в терминале вашей IDE:

### 1. Клонирование репозитория
Склонируйте проект и перейдите в его корневую папку:
git clone https://github.com/ZoeKrasnitskaia/QA_Less.git
### 2. Настройка окружения и установка зависимостей
python -m venv venv 
pip install -r requirements.txt
cp .env.example .env
### 3. Запуск автоестов
pytest -q
